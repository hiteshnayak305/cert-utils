# cert-utils
SSL cert utilities

- Copy [secret.example.sh](secret.example.sh) to secret.sh in root directory
- Change PASS variable to some secret string
- Commands
  - generateRootKey.sh
  - generateRootCACert.sh
  - renewRootCACert.sh
  - generateDomainKey.sh
  - generateDomainCert.sh
  - renewDomainCert.sh
